# GI-ReactW3 - React Week 3 Challenges

This repository contains three React applications built with Vite, demonstrating proficiency in React hooks and React Router.

## Projects

### Easy - Counter App
A simple counter application using React hooks.

**Features:**
- Increment counter
- Decrement counter
- Uses `useState` hook

**To run:**
```bash
cd Easy/counter-app
npm install
npm run dev
```

---

### Medium - Movie Search App
A movie search application that fetches data from the TMDB API.

**Features:**
- Search for movies by title
- Display movie results with posters and ratings
- Click on a movie to view detailed information
- Navigate between search page and detail page using React Router
- Uses `useState`, `useEffect`, and React Router

**To run:**
```bash
cd Medium/movie-search
npm install
npm run dev
```

**Note:** Requires TMDB API key. Create a `.env` file with:
```
VITE_TMDB_API_KEY=your_api_key_here
```

---

### Hard - Todo List App
A task management application with full CRUD functionality.

**Features:**
- Add new tasks
- Delete tasks
- Edit task details
- Navigate between task list and task detail pages using React Router
- Uses `useState` and React Router

**To run:**
```bash
cd Hard/todo-list
npm install
npm run dev
```
